测距仪客户端

在线安装

当前版本 1.0.0.9 更新日期 2014/10/22


安装

  1. 点击“在线安装”
  2. 在打开的页面下方点击“安装”按钮，浏览器会下载安装程序
  3. 运行安装程序，按照提示安装

运行

  1. 将COM转USB连接线分别接入笔记本和测距仪
  2. Windows 7系统：开始 -> 所有程序 -> SGIDI -> 测距仪客户端
  3. 首先在下拉列表里选择端口号。不知道端口号？看这里
  4. 然后根据实际需求点击“调试”或者“采集”按钮

如何确定端口号

  1. Windows 7系统：开始 -> 控制面板 -> 设备管理器
  2. 在列表里找到并展开“端口”就能看到端口号，例如COM3

调试

  1. 选择好端口号后，点击“调试”按钮
  2. 输入测距仪编号
  3. 点击“测量”按钮，等待大约3秒，就能看到结果
  4. 点击“打开激光”按钮，等待大约3秒，测距仪进入激光状态
  5. 点击“关闭激光”按钮，退出激光状态
  6. 关闭此窗口就可回到初始界面

采集

  1. 选择好端口号后，点击“调试”按钮
  2. 首先点击“读取点号”按钮，选择点号文件，等待程序读取。读取成功时会显示已读取的点数。参看点号文件格式
  3. 然后点击“开始采集”按钮，程序开始不断地测量，最近一次测量结果会显示在界面上
  4. 点击“停止采集”按钮即可停止测量
  5. 点击“保存”按钮将本轮测量的结果保存到文件中。文件名默认是当前时间，例如201412221106.txt。参看结果文件格式
  6. 关闭此窗口就可回到初始界面

点号文件格式

范例下载

点号文件应为文本文件，每一行代表一个点，包含两个数据，第一个是点名，第二个是对应的测距仪编号；这两个数据之间用英文逗号分隔

例如

60环,20
72环,22

结果文件格式

范例下载

结果文件为文本文件，每一行代表一次测量结果，包含两个数据，第一个是点名，第二个是距离，单位为米；这两个数据之间用英文逗号分隔

例如

60环,2.848
72环,2.850
